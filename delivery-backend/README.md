# Delivery Management System - Backend

This is the Node.js / Express / TypeScript backend for the Delivery Management System assignment. It provides role-based authentication (buyer, seller, admin), order lifecycle management, and real-time updates via Socket.io.

See the frontend README in `../delivery-frontend/README.md` for the client application.

Quickstart

- Copy `.env.example` to `.env` and fill values
- Install dependencies: `npm install`
- Run dev server: `npm run dev`

API endpoints and usage are described in the project README.
