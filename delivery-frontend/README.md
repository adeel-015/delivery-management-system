# Delivery Management System - Frontend

React + Vite + TypeScript frontend for the Delivery Management System. Uses Tailwind CSS and Socket.io client to receive real-time updates.

Quickstart

1. Install: `npm install`
2. Run dev server: `npm run dev`

The app proxies `/api` to `http://localhost:5001` by default. Update `vite.config.ts` if needed.
